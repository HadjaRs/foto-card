# ✨ My Social Links Card ✨

Bem-vinda ao meu cantinho! Este é o meu cartão de visitas digital, feito com muito carinho para centralizar redes sociais em um só lugar. 🎀

### 🔗 Acesse aqui:
> **[Hadjars - Rede Card](https://hadjars.github.io/foto-card/)**

---

### ☁️ Sobre o Projeto

* **HTML5** (Estrutura)
* **CSS3** (Gradients & Animations)
* **JavaScript** (Interatividade)
* **PWA** (Para você instalar no seu celular como um app! 📱)

### 💅 Funcionalidades
- [x] Botão interativo com efeito toggle
- [x] Background animado em tons de rosa e roxo
- [x] Design responsivo 
- [x] Offline ready (graças ao Service Worker)

---

### 🛠 Como usar?
Se quiser usar como base para o seu:
1. Faça um **Fork** deste repositório
2. Troque a imagem na pasta `img/`
3. Ajuste os links no `index.html`
4. Seja feliz! ✨
